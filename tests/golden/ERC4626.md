## Overview

ERC4626 is a minimal tokenized Vault implementation that mints and burns ERC20 shares representing claims on an underlying asset (see tests/fixtures/tier0_erc4626/src/tokens/ERC4626.sol:34-40 and 46-58). It stores an immutable ERC20 asset and implements the standard deposit/mint and withdraw/redeem operations with safe transfers via SafeTransferLib (lines 34-40, 46-58, 73-93, 95-116). Conversion and preview helpers compute share/asset amounts using totalSupply and totalAssets(), deferring the actual totalAssets implementation to subclasses (lines 122-154). Hooks beforeWithdraw and afterDeposit are provided for extensions to override (lines 180-182).
